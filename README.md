
# MongoDb with Nodejs
This boilerplate will help you to kick start your MongoDb project

## Prerequisite
MongoD should be running on default port 27017

## Run Locally

Clone the project

```bash
  git clone https://github.com/ErSachinVashist/mongoboilerplate.git
```

Go to the project directory

```bash
  cd mongoboilerplate
```

Install dependencies

```bash
  yarn install
```

Create mock database

```bash
  yarn mock
```

Start nodejs server

```bash
  yarn start
```
